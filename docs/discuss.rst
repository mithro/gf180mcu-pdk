*******
Discuss
*******

Questions, ideas, and help with the GF180MCU PDK are welcome across the open
source silicon community. There are a few places to chat:

FOSSi Chat (Matrix)
===================

The `#gf180mcu:fossi-chat.org <https://matrix.to/#/#gf180mcu:fossi-chat.org>`__
room is the main place to discuss the GF180MCU open PDK and process. It lives
on `FOSSi Chat <https://fossi-chat.org>`__, the Matrix server run by the FOSSi
Foundation.

wafer.space Discord
===================

The `wafer.space Discord <https://discord.gg/43y2t53jpE>`__ is the place to get
help with wafer.space GF180MCU runs and to chat with other designers taping out
on the process.

Tiny Tapeout Discord
====================

The `Tiny Tapeout Discord <https://discord.com/invite/qZHPrPsmt6>`__ is the
place to discuss the GF180MCU Tiny Tapeout shuttles, in the GF180MCU shuttle
channel.
